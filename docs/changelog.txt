Changelog

* 0.9.2 *
Added Freshney Freight Barge (for inland waterways).  12mph, 45t of cargo.
Added Labrador Utility Vessel.  18mph, 30 passengers or 12t of cargo.
Added Josephine Utility Vessel.  16mph, 89 passengers or 52t of cargo.
Added Sungei Catamaran.  Available 1997, 46mph, 250 passengers or 64 bags of mail. (graphics: Emperor Jake, based on DanMacK sprite)
Improved appearance of buy menu: ships are center-aligned.
Fixed freight / mail capacities for hydrofoils.
Fixed lighting for several ships.
Internal changes to move from pcx to png files.

* 0.9.1 *
Added Fitzroy Utility Tug (small).
Improved appearance of medium utility fishboat.
Improved appearance of Tyskebukta Freighter and Rockall Freighter (lighting was wrong).
Fixed Porcupine Hovercraft costs - was too expensive to buy, too cheap to run.
Fixed Log Raft running costs.
Fixed Paddle Steamer costs (were too high) and PAX capacity (was too low).
Fixed offsets and running costs for Hydrofoils.
Fixed buy menu offsets for multiple ships.
Adjusted costs for Ferries to be fairer.
Internal changes to move from pcx to png files; refactored CPP defines for strings.

* 0.9.0 *
Named all ships.  Names drawn from two sources, cookies awarded to the first person to correctly guess both. 
Added Fiumelatte Paddle Steamer.  Available 1870, 23mph top speed.  (graphics: Lead@inbox, DanMacK)
Added Klyazma Hydrofoil.  Available 1961, 40mph top speed.  (graphics: Lead@inbox, DanMacK)
Added Xacuabs 818 Hydrofoil.  Available 1975, 56mph top speed.  (graphics: Lead@inbox, DanMacK)
Added Cahaba Towboat.  Available 1870, 15mph, refits to various capacities.  (graphics: DanMacK)	
Increased size of Small Coaster (Oran Freighter) graphics to better match capacity.
Additional cranes for Large Coaster (Patraikos Freighter).
Improved appearance of Large Trader (Tyskebukta Freighter).
Improved appearance of Small Trader (Rockall Freighter).
Match size of Log Raft graphics to refitted capacity.
Fixed all offsets for log rafts, closing one bug and generally making them better.  

* 0.8.2 *
Many ships now have a smaller capacity for freight than for passengers.
Utility Vessel passenger capacity increased to 60.
FISH no longer disables default ferries - this is a temporary feature
until more small-sized ferries are finished.
Various minor graphical fixes.

* 0.8.1 *
Improved graphical style of Small and Large Coasters.
Add cranes / masts to some other ships.
Log Rafts can be refitted to multiple capacities (95t, 210t, 400t).
Internal changes (convert more pcx files to png, tidy up a few things).

* 0.8.0 *
Restored compatibility with OpenTTD 0.7.x and 1.0.x (was incompatible due to use smoke, now resolved).
Improved graphical style of many ships, including complete rework of River Boats.
Several changes to improve gameplay balance of set:
- removed Medium Coaster, increased capacity of Small Coaster.
- removed Large Trader, renamed Medium Trader to Large Trader. 
- changed capacity of several other freight ships (some bigger, some smaller).
Changes shouldn't break savegames, but there may be a few odd effects.

* 0.7.0 *
Cargo hold graphics for River Boats when loading / unloading.
Utility Vessel (Medium) uses ferry graphics when refitted to PAX (Dan MacK).
Smoke for some ships (needs OpenTTD r21240 or newer).
Updated minimum OpenTTD version requirements (Planetmaker).
Many internal changes to make the set easier to develop and maintain.
